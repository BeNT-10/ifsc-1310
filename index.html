Progressive Enhancement Writing Assignment

Progressive enhancement is one of the many web design strategies that helps web developers keep their work relevent in a rapidly changing ecosystem. Its basic premise states that web developers should design their sites in layers. The first layer is the content layer - one that will be visible to everyone, on every browser. After this layer has been established, a presentation layer should be built on top of it. This layer usually consists solely of CSS, and only dictates the the way the content layer is presented to the user. An initial first draft of the presentation layer should be made of features that will be visible across all browsers. Finally, a user experience layer that surrounds both the content and presentation layers should be made. This layer typically consists of JavaScript functionality, but includes any and all features that turn a website from a piece of reading material into an "experience."

After the first iteration of all the website layers is completed, there should exist a website that can be viewed by any user on any browser. It is now time to "progressively enhance" it. There are many ways to go about this, but the suggested method is to use feature detection to incrementally add functionality across all layers. This is done until the web developer decides that they are happy with the website, or that the website contains the "best" features across all browsers and offers a tailored experience to everyone.
